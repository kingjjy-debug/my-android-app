# App Name
대한항공 마일리지 모으기

# Summary
대한항공 마일리지로 전환 가능한 주요 포인트 적립/전환 경로를 한 화면에 모아, 매일 접속하고 완료 여부를 기록할 수 있는 앱

# Features
- 무료 적립형 포인트사(네이버페이, OK캐시백, L.POINT)는 두 개의 버튼 제공:
  1) "포인트 적립하기" → 해당 포인트사의 적립/이벤트 페이지
  2) "마일리지 전환하기" → 대한항공 마일리지 전환 안내 페이지
- 카드사 포인트사(신한, 삼성, KB, 현대)는 "마일리지 전환하기" 버튼만 제공
- 각 항목에 "완료" 토글(체크박스 또는 스위치). 완료 시 굵은 초록색 텍스트 "일일미션 수행 완료!" 표시
- 완료 상태는 로컬에 저장되고 앱 재실행 시 유지
- 매일 0시 기준으로 완료 상태 자동 초기화

# Target Users
- 대한항공 마일리지를 꾸준히 모으고 싶은 사용자
- 무료 포인트 적립 + 카드 포인트 전환을 한 앱에서 관리하려는 사용자

# Technical Notes
- 데이터: 코드 내 상수 배열로 하드코딩
- 권한: 없음
- 네트워크: 외부 링크 연결 시만 사용
- 상태 저장: SharedPreferences (항목별 완료 여부 + 마지막 기록 날짜)
- 리셋 규칙: 앱 실행 시 오늘 날짜가 마지막 기록 날짜와 다르면 모든 완료 상태 초기화

# UI/UX Requirements
- 단일 화면(메인). 상단 앱바 제목: "대한항공 마일리지 모으기"
- 스크롤 가능한 카드 리스트. 각 카드 구성:
  - 상단: 포인트사 이름(굵게)
  - 버튼 행: [포인트 적립하기] [마일리지 전환하기] (무료 적립형만) / [마일리지 전환하기] (카드사)
  - 우측 또는 하단: "완료" 토글 (ON → "일일미션 수행 완료!" 초록색 표시)
- 버튼 라벨은 한국어, 최소 48dp 터치 영역
- Material 3 디자인, 라이트/다크 모드 대응
- 네트워크 실패 시 토스트 안내: "연결할 수 없습니다. 네트워크 상태를 확인하세요."

# Non-Goals
- 대한항공 API 연동 없음
- 앱 내부에서 포인트 적립/전환 처리 없음 (외부 링크 연결만)

# Acceptance Criteria
- 앱 실행 시 7개 카드(네이버페이, OK캐시백, L.POINT, 신한, 삼성, KB, 현대)가 표시됨
- 무료 적립형 3개는 [포인트 적립하기], [마일리지 전환하기] 버튼 모두 있음
- 카드사 4개는 [마일리지 전환하기] 버튼만 있음
- 모든 카드에 "완료" 토글이 있으며, ON → "일일미션 수행 완료!" 표시
- 완료 상태는 앱 재실행 후에도 유지되고, 날짜 변경 시 자동 초기화됨
- 권한 요청 다이얼로그가 뜨지 않음
